Plugin adds ability to create automatic coupons with additional features.

== Description ==

One of the best marketing practices is gift giving - it will make your Customers come back to your shop much more often! This [coupon plugin](http://beecommerce.pl/darmowe-wtyczki/) adds ability to create [various promotions](http://beecommerce.pl/darmowe-wtyczki/) using automatic coupons for WooCommerce with some additional features:

*   "cheapest products for free" - specified number of cheapest products in the cart get discounted
*   "buy x get y" - add free products of the same type to cart
*   "gift" - adds specified product to cart for free

== How it works? ==

1. After successful plugin activation create a new coupon using WooCommerce 'Coupons' screen - a new discount type 'Free product' should be available.
2. Use 'Free Product settings' tab to configure coupon details - choose the type of discount, how many matching products in cart will activate the coupon, how many free products should be added to cart and ID of the product that will be added to customer's cart as a gift (only in case of 'Gift' type of discount).
3. The coupon will be **automatically** used upon cart contents matching coupon settings.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create new coupon using WooCommerce 'Coupons' screen - a new discount type 'Free product' should be available.
4. Use 'Free Product settings' tab to configure coupon details.
